# RingDNA

RingDNA (now Revenue.io) is an intelligent revenue platform providing REST APIs for sales dialing, call recording, conversation analytics, real-time coaching, and CRM integration for revenue teams. The platform is 100% Salesforce-native and includes the RingDNA Dialer, Moments real-time AI guidance, Guided Selling cadences, and full Conversation Intelligence.
